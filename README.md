## 项目一：在线零售销售表现与客户价值分析

本项目基于在线零售交易数据，围绕销售表现和客户价值展开分析。通过 Excel 完成数据清洗、字段构造、数据透视表分析和图表制作，分析销售额、订单数、客单价、月度销售趋势、国家销售贡献、热销商品和客户复购情况，并基于 RFM 模型完成客户分层。

### 核心分析内容

- 构造销售金额、订单月份、是否取消订单等分析字段
- 分析整体销售额、订单数、客户数和客单价
- 分析月度销售趋势和国家销售贡献
- 识别销售额 Top10 和销量 Top10 商品
- 分析客户消费金额、订单数和复购结构
- 基于 RFM 模型完成客户分层，并提出运营建议

### 项目收获

通过该项目，我熟悉了 Excel 数据透视表、字段构造、销售分析指标和客户分层分析方法，完成了从原始交易数据到业务分析报告的完整流程。

## 项目二：淘宝用户行为分析

本项目基于天池淘宝用户行为数据集，抽取约 100 万条用户行为记录进行分析。使用 SQLite 完成数据导入、时间字段处理和 SQL 指标统计，并使用 Excel 制作可视化图表，分析用户活跃趋势、行为结构、转化漏斗、复购情况以及商品和类目购买偏好。

### 核心分析内容

- 使用 SQL 统计行为总量、用户数、商品数和类目数
- 分析用户行为类型占比，识别浏览、收藏、加购、购买行为结构
- 分析每日 PV / UV 趋势和各小时用户行为分布
- 构建浏览 → 收藏/加购 → 购买的转化漏斗
- 分析购买用户复购率和用户留存情况
- 识别商品购买次数 Top10 和类目购买次数 Top10

### 核心结论

- 浏览行为占比最高，达到 89.60%，购买行为占比为 2.04%
- 浏览到收藏/加购的转化率为 9.33%，浏览到购买的整体转化率为 2.28%
- 购买用户复购率为 66.07%，说明购买用户具有较好的复购基础
- 用户活跃高峰集中在中午至下午时段，适合进行重点活动曝光和运营触达
- 类目维度比单个商品维度更能体现用户购买偏好

### 项目收获

通过该项目，我练习了 SQL 查询、条件聚合、去重统计、转化漏斗分析、复购分析、留存分析和 Excel 可视化表达，进一步理解了电商用户行为分析的基本思路。
